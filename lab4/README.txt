Write two programs for processing text files. One of them performs
line-by-line, the other one character processing. The source file must
be created using any text editor.
1. A program that processes a text file and writes the processed
data to a file with the same name, but with a different type.
Leave strings that don't contain numbers.
Command line parameters:
	- Input file name
	- Number of strings processed
2. A program that performs character-based processimg of a text
file.
Insert '\n' after each point.
Command line parameters:
	- Input file name
	- Number of inserts
